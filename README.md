# MakNews

Merupakan aplikasi self project dengan menggunakan web service https://newsapi.org/


# Technologies
- Xcode 13.2.1
- Swift
- UIKit
- VIPER
- Programmatic UI
- Alamofire
- KingFisher
- NVActivityIndicatorView
- UIScrollView-InfiniteScroll

# Features
- Top headline berita
- Kategori berita
- List sumber berita
- List artikel berita
- Tampilan web artikel
- Pencarian berdasarkan artikel dan sumber berita
